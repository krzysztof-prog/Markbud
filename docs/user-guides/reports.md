# Raporty - Przewodnik Użytkownika

Przewodnik generowania raportów i eksportów w systemie AKROBUD.

---

## Dostępne Raporty

### 1. Raporty Miesięczne

**Lokalizacja:** Zestawienia → Raporty miesięczne

**Typy raportów:**
- **Zlecenia** - podsumowanie zleceń w miesiącu
- **Dostawy** - realizowane dostawy
- **Obroty** - wartość sprzedaży
- **Magazyn** - zużycie profili

**Generowanie:**
1. Wybierz miesiąc i rok
2. Wybierz typ raportu
3. Kliknij "Generuj PDF"
4. Pobierz plik

### 2. Raporty Zleceń

**Export listy zleceń:**
1. Zlecenia → Ustaw filtry (status, okres)
2. Kliknij "Eksportuj"
3. Format: Excel / CSV / PDF

**Raport szczegółowy zlecenia:**
1. Otwórz zlecenie
2. Kliknij "Drukuj"
3. PDF zawiera: dane, okna, zapotrzebowanie

### 3. Protokoły Dostawy

**Automatyczne generowanie:**
1. Dostawy → Wybierz dostawę
2. Kliknij "Generuj protokół"
3. PDF z: zleceniami, oknami, layoutem palet

### 4. Raporty Magazynowe

**Stan magazynu:**
- Magazyn → Raporty → "Stan magazynu"
- Pełna lista profili, stany, wartości
- Export: Excel / PDF

**Niedobory:**
- Magazyn → Raporty → "Niedobory"
- Lista profili do zamówienia
- Sortowane po pilności

**Rotacja profili:**
- Magazyn → Statystyki → "Raport rotacji"
- Które profile szybko się zużywają
- Optymalizacja zamówień

---

## Eksporty Danych

### Excel Export
Wszystkie tabele można eksportować do Excel:
- Kliknij ikonę ⬇️ (Export)
- Wybierz "Excel"
- Otwórz plik .xlsx

### CSV Export
Dla importu do innych systemów:
- Export → "CSV"
- Kodowanie: UTF-8
- Separator: przecinek

### PDF Export
Do druku i archiwizacji:
- Export → "PDF"
- Gotowy do druku
- Zawiera logo firmy

## FAQ - Raporty

**Q: Jak często generować raporty miesięczne?**
A: Na koniec każdego miesiąca do rozliczeń.

**Q: Czy mogę dostosować szablon protokołu dostawy?**
A: Tak, Ustawienia → Protokoły → Edytuj szablon.
