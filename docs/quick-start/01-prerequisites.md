# Prerequisites

Przed rozpoczeciem upewnij sie, ze masz zainstalowane wymagane narzedzia.

## Wymagane

| Tool | Minimalna Wersja | Sprawdz | Instalacja |
|------|------------------|---------|------------|
| **Node.js** | 18.0.0 | `node --version` | [nodejs.org](https://nodejs.org/) |
| **pnpm** | 8.0.0 | `pnpm --version` | `npm install -g pnpm` |
| **Git** | 2.30.0 | `git --version` | [git-scm.com](https://git-scm.com/) |

## Opcjonalne (ale zalecane)

- **VS Code** - zalecany editor
  - Extensions: Prisma, ESLint, Prettier, Tailwind CSS IntelliSense
- **Prisma Studio Desktop** - GUI dla bazy danych

## Weryfikacja instalacji

Uruchom ponizsze komendy aby sprawdzic czy wszystko jest zainstalowane:

```bash
node --version    # Powinno zwrocic v18.x.x lub wyzej
pnpm --version    # Powinno zwrocic 8.x.x lub wyzej
git --version     # Powinno zwrocic 2.30.x lub wyzej
```
